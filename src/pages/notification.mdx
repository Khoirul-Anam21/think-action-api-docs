# notification

notifications are how User received informations triggered by other user

## Goal notification model

The notification model contains all the information about notification

### Properties

<Properties>
  <Property name="_id" type="objectId">
    The unique identifier for the notification.
  </Property>
  <Property name="user_id" type="objectId">
    The unique identifier of the user receiving the notification.
  </Property>
  <Property name="user_notifier_id" type="objectId">
    The unique identifier of the user who triggered the notification.
  </Property>
  <Property name="post_id" type="objectId">
    The unique identifier of the related post (if applicable).
  </Property>
  <Property name="message" type="string">
    The content of the notification message.
  </Property>
</Properties>
