# goals

goal is the sub part of the resolution that contains the materials about the resolution. this is the part where users actually learn the resolution.

## The goal model

The goal model contains all the information about the goals.

### Properties

<Properties>
  <Property name="id" type="string">
    Unique identifier for the Goal.
  </Property>
  <Property name="resolution_id" type="string">
    Identifier of the Resolution associated with the Goal.
  </Property>
  <Property name="goal" type="string">
    Text goal of the Goal.
  </Property>
  <Property name="category_id" type="string">
    Identifier of the category to which the Goal belongs.
  </Property>
  <Property name="shareType" type="string">
    Type of sharing for the Goal (e.g., public, private).
  </Property>
  <Property name="imageUrl" type="string">
    URL of the image associated with the Goal.
  </Property>
  <Property name="dueDate" type="string">
    Due date for the Goal.
  </Property>
</Properties>

## Create a goal {{ tag: 'POST', label: '/v1/goals' }}

<Row>
  <Col>

    This endpoint allows you to retrieve a detailed information about the goal. Using goal id as http params. And user must specify the resolution id in the body.

    ### Required attributes

    <Properties>
      <Property name="goal" type="string">
        Text goal of the Goal.
      </Property>
      <Property name="category_id" type="string">
        Identifier of the category to which the Goal belongs.
      </Property>
      <Property name="shareType" type="string">
        Type of sharing for the Goal (e.g., public, private).
      </Property>
      <Property name="images" type="array">
        images associated with the Goal.
      </Property>
      <Property name="dueDate" type="string">
        Due date for the Goal.
      </Property>
    </Properties>



  </Col>
  <Col sticky>

    <CodeGroup title="Request" tag="POST" label="/v1/goals">


    ```js
    const response = await axios.post('/v1/goals', {
      goalText: 'Complete Chapter 1',
      category_id: 'category123',
      shareType: 'public',
      imageUrl: 'https://example.com/image.jpg',
      dueDate: '2023-06-30'
    });
    ```

    </CodeGroup>

    ```json {{ title: 'Response' }}

      "acknowledged": true
    }
    ```

  </Col>
