# goals

Course content is the sub part of the course that contains the materials about the course. this is the part where users actually learn the course.

## The course content model

The goal model contains all the information about the goals.

### Properties

<Properties>
  <Property name="id" type="string">
    Unique identifier for the Goal.
  </Property>
  <Property name="resolution_id" type="string">
    Identifier of the Resolution associated with the Goal.
  </Property>
  <Property name="goalText" type="string">
    Text content of the Goal.
  </Property>
  <Property name="category_id" type="string">
    Identifier of the category to which the Goal belongs.
  </Property>
  <Property name="shareType" type="string">
    Type of sharing for the Goal (e.g., public, private).
  </Property>
  <Property name="imageUrl" type="string">
    URL of the image associated with the Goal.
  </Property>
  <Property name="dueDate" type="string">
    Due date for the Goal.
  </Property>
</Properties>


## The assignment model

The assignment model contains the information of the assignment

### Properties

<Properties>
  <Property name="_id" type="string">
    Unique identifier for the goal.
  </Property>
  <Property name="title" type="string">
    The title of course content.
  </Property>
  <Property name="instruction" type="string">
    The instruction of the assignment
  </Property>
  <Property name="contentId" type="string">
    Referring to the content material of the course.
  </Property>
  <Property name="submissions" type="array">
    The student assignment's submission
  </Property>
  <Property name="posted" type="timestamp">
    Timestamp of when the assignment was created.
  </Property>
</Properties>
