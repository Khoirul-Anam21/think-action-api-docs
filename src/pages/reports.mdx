# reports

reports are how student or facilitator analize or obtain information about their activity in this LMS {{ className: 'lead' }}

## Student report model 

The report model contains all the learning informations and progresses.

### Properties

<Properties>
  <Property name="id" type="string">
    Unique identifier for the report.
  </Property>
  <Property name="user_id" type="string">
    Unique identifier for the user associated with the report.
  </Property>
  <Property name="course_id" type="string">
    Unique identifier for the course associated with the report
  </Property>
  <Property name="progress" type="double">
    The percentage of course completion.
  </Property>
  <Property name="score" type="string">
    The score report of the course.
  </Property>
  <Property name="status" type="integer">
    The completion status of the course.
  </Property>
  <Property name="latest_update" type="timestamp">
    Timestamp of when the report was updated.
  </Property>
</Properties>

## Facilitator report model

The report model contains all the information about courses informations that the facilitator has.

### Properties

<Properties>
  <Property name="id" type="string">
    Unique identifier for the report.
  </Property>
  <Property name="user_id" type="string">
    Unique identifier for the message associated with the report.
  </Property>
  <Property name="total_course" type="string">
    Total courses the facilitator have
  </Property>
  <Property name="total_participant" type="string">
    Total partipants enrolled in the facilitator courses.
  </Property>
  <Property name="complete_participant" type="string">
    Total participants completed the facilitator courses
  </Property>
  <Property name="latest_update" type="timestamp">
    Timestamp of when the report was updated.
  </Property>
</Properties>
