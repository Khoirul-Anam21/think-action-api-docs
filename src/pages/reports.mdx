# Report

reports are how User analize their goal achievements

## Goal report model

The report model contains all the information about goal achievements

### Properties

<Properties>
  <Property name="id" type="string">
    Unique identifier for the report.
  </Property>
  <Property name="user_id" type="string">
    Unique identifier for the message associated with the report.
  </Property>
  <Property name="year" type="number">
  Year indicator
  </Property>
  <Property name="month" type="number">
    Month indicator
  </Property>
  <Property name="week" type="number">
    Week indicator
  </Property>
  <Property name="week completion" type="array">
    Completion of goal every week by categories
  </Property>
</Properties>
