# Users

As the name suggests, users are a core part of Protocol — Making new account. We'll look at how to query, create, update, and delete users. {{ className: 'lead' }}

## The user model

The user model contains all the information about your users, such as their username, email, role, and profile photo. It also able to edit account information.

### Properties

<Properties>
  <Property name="id" type="string">
    Unique identifier for the user.
  </Property>
  <Property name="username" type="string">
    The username for the user.
  </Property>
  <Property name="email" type="string">
    The email for the user.
  </Property>
  <Property name="password" type="string">
    The password of the user.
  </Property>
  <Property name="photo" type="string">
    The Url of the profile picture
  </Property>
</Properties>

## Sign up {{ tag: 'POST', label: '/v1/auth/signup' }}

<Row>
  <Col>

    This endpoint allows you to add a new user. To add a user, you must provide email, username, and password.

    ### Required attributes

    <Properties>
      <Property name="username" type="string">
        The username for the user.
      </Property>
      <Property name="email" type="string">
        The email for the user.
      </Property>
      <Property name="password" type="string">
        The password for the user.
      </Property>
    </Properties>


  </Col>
  <Col sticky>

    <CodeGroup title="Request" tag="POST" label="/v1/auth/signup">

    ```js
    const postResponse = await axios.post('https://api.example.com/v1/auth/signup', {
      username: 'John',
      email: 'john@example.com',
      password: 'admin'
    });
    ```

    </CodeGroup>

    ```json {{ title: 'Response' }}
    {
      "id": "WAz8eIbvDR60rouK",
      "acknowledged": true,
    }
    ```

  </Col>
</Row>
