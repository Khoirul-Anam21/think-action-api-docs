# Cheers

Cheer is the logic of liking a post

## The cheer models

The Cheer model contains all the information about the Cheers.

### Properties

<Properties>
  <Property name="id" type="string">
    Unique identifier for the Cheer.
  </Property>
  <Property name="post_id" type="string">
    Identifier of the post associated with the Cheer.
  </Property>
  <Property name="postType" type="string">
    Type of the post for performance optimization.
  </Property>
  <Property name="userCheerer_id" type="string">
    User cheer
  </Property>
</Properties>

---

## List all Cheers {{ tag: 'GET', label: '/v1/cheers' }}

<Row>
  <Col>

    This endpoint allows you to retrieve a paginated list of all your Cheers. This endpoint usually used for catalog. By default, a maximum of ten Cheers are shown per page.

    ### Required attributes

    <Properties>
      <Property name="post_id" type="string">
        id of the post refer to the cheer
      </Property>
      <Property name="post_type" type="string">
        type of the post
      </Property>
    </Properties>

  </Col>
  <Col sticky>

    <CodeGroup title="Request" tag="GET" label="/v1/cheers?postId='...'&postType='...'">

    ```js
      const response = await axios.get('/v1/cheers', params: {
        postId: 'sdfapoifw',
        postType: 'resolution',
      },);
    ```

    </CodeGroup>

    ```json {{ title: 'Response' }}
    
    {
      "cheerers": [
        {
          "_id": "cheer123",
          "photo": "http://example.com/image.jpg",
          "accountName": "Anam",
          "supporter": true
        },
        {
          "id": "hSIhXBhNe8X1d8Et"
          // ...
        }
      ],
      "pagination": {
        "page": 1,
        "pageSize": 1,
        "pageCount": 4,
        "totalDocument": 16
      }
    }
    ```

  </Col>
</Row>


## Create a Cheer {{ tag: 'POST', label: '/v1/cheers' }}

<Row>
  <Col>

    This endpoint allows you to add a Cheer. user must logged in

    ### Required attributes

<Properties>
  <Property name="post_id" type="string">
    Identifier of the post associated with the Cheer.
  </Property>
  <Property name="postType" type="string">
    Type of the post for performance optimization.
  </Property>
</Properties>

  </Col>
  <Col sticky>

    <CodeGroup title="Request" tag="POST" label="/v1/cheers">

    ```js
      const response = await axios.post('/v1/cheers', {
        post_id: 'user123',
        postType: 'resolution',
      });
    ```

    </CodeGroup>

    ```json {{ title: 'Response' }}
    {
      "id": "xgQQXg3hrtjh7AvZ",
      "acknowledged": true,
    }
    ```

  </Col>
</Row>

## Delete a Cheer {{ tag: 'DELETE', label: '/v1/cheers/:id' }}

<Row>
  <Col>

    This endpoint allows you to delete a Cheer by providing the Cheer id. Deleting this Cheer will delete all the comments inside.

  </Col>
  <Col sticky>

    <CodeGroup title="Request" tag="DELETE" label="/v1/cheers/xgQQXg3hrtjh7AvZ">

    ```js
    const response = await axios.delete('/v1/cheers/cheer123');
    ```

    </CodeGroup>

    ```json {{ title: 'Response' }}
    // empty response
    ```

  </Col>
</Row>
