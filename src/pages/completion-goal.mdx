# Goal Completion

This Goal Completion contains some information that's not really involved in the business logic but important however. {{ className: 'lead' }}

## Completion model

The attachment model contains all the information about the role type of the user. Decide whether the user is a student or facilitator.

### Properties

<Properties>
  <Property name="id" type="string">
    Unique identifier for the Entity.
  </Property>
  <Property name="user_id" type="string">
    Identifier of the User associated with the Entity.
  </Property>
  <Property name="goal_id" type="string">
    Identifier of the Goal associated with the Entity.
  </Property>
  <Property name="completion_id" type="string">
    Identifier of the completion to which the Entity belongs.
  </Property>
  <Property name="caption" type="string">
    Caption or description of the Entity.
  </Property>
  <Property name="image" type="string">
    URL or path of the image associated with the Entity.
  </Property>
  <Property name="isCompleteResolution" type="boolean">
    Flag indicating whether the resolution is complete for the Entity.
  </Property>
</Properties>
