# Categorys

Category for the resolution, embedded in user data. Max 7 per user

## The category models

The Category model contains all the information about the Categorys.

### Properties

<Properties>
  <Property name="id" type="string">
    Unique identifier for the Category.
  </Property>
  <Property name="category" type="string">
    Category name
  </Property>
</Properties>

## List all Categories {{ tag: 'GET', label: '/v1/categories' }}

<Row>
  <Col>

    This endpoint allows you to retrieve a paginated list of all your Categorys. This endpoint usually used for catalog. By default, a maximum of ten Categorys are shown per page.


  </Col>
  <Col sticky>

    <CodeGroup title="Request" tag="GET" label="/v1/categories">

    ```js
      const response = await axios.get('/v1/categories');
    ```
