# Resolutions

Resolution is the main product of this application. Resolution is the topic of the learning that contains many contents{{ className: 'lead' }}

## The resolution models

The Resolution model contains all the information about the Resolutions.

### Properties

<Properties>
  <Property name="id" type="string">
    Unique identifier for the Resolution.
  </Property>
  <Property name="user_id" type="string">
    Identifier of the user associated with the Resolution.
  </Property>
  <Property name="resolution" type="string">
    Text content of the Resolution.
  </Property>
  <Property name="imageUrl" type="string">
    URL of the image associated with the Resolution.
  </Property>
  <Property name="category_id" type="string">
    Identifier of the category to which the Resolution belongs.
  </Property>
  <Property name="shareType" type="string">
    Type of sharing for the Resolution (e.g., public, private).
  </Property>
  <Property name="completed" type="boolean">
    Indicates whether the Resolution is completed or not.
  </Property>
  <Property name="dueDate" type="string">
    Due date for the Resolution.
  </Property>
</Properties>

## Create a Resolution {{ tag: 'POST', label: '/v1/resolutions' }}

<Row>
  <Col>

    This endpoint allows you to add a Resolution. 

    ### Required attributes

    <Properties>
      <Property name="user_id" type="string">
          Identifier of the user associated with the Resolution.
        </Property>
        <Property name="resolution" type="string">
          Text content of the Resolution.
        </Property>
        <Property name="imageUrl" type="array">
          URL of the images associated with the Resolution.
        </Property>
        <Property name="category_id" type="string">
          Identifier of the category to which the Resolution belongs.
        </Property>
        <Property name="shareType" type="string">
          Type of sharing for the Resolution (e.g., public, private).
        </Property>
        <Property name="dueDate" type="string">
          Due date for the Resolution.
        </Property>
    </Properties>
    

  </Col>
  <Col sticky>

    <CodeGroup title="Request" tag="POST" label="/v1/resolutions">

    ```js
      const response = await axios.post('/v1/resolutions', {
        user_id: 'user123',
        resolution: 'Finish reading a book',
        imageUrl: ['<image file>', ...],
        category_id: 'category123',
        shareType: 'public',
        dueDate: '2023-05-31'
      });
    ```

    </CodeGroup>

    ```json {{ title: 'Response' }}
    {
      "id": "xgQQXg3hrtjh7AvZ",
      "acknowledged": true,
    }
    ```

  </Col>
</Row>
